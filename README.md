# CBR Currency Tracker

Python pet-проект для портфолио: загрузка официальных курсов валют ЦБ РФ, сохранение истории в PostgreSQL и просмотр курсов USD, EUR и CNY через консольное меню.

## Возможности

- Загрузка курсов за сегодня
- Загрузка курсов на конкретную дату
- Сохранение данных в PostgreSQL
- Просмотр последних сохранённых курсов
- Просмотр курсов из базы на выбранную дату
- Расчёт среднего курса за последние 3 полных месяца
- Автоматическая подгрузка исторических данных по диапазону дат

## Стек

- Python 3
- PostgreSQL
- requests
- psycopg2-binary

## Установка

1. Клонировать репозиторий:

```bash
git clone https://github.com/lepixa/cbr-currency-tracker.git
cd cbr-currency-tracker
```

2. Установить зависимости:

```bash
py -m pip install -r requirements.txt
```

3. Настроить подключение к PostgreSQL в `src/config.py`.

## Запуск

### Основное меню

```bash
py -u main.py
```

### Загрузка данных за последние 3 полных месяца

```bash
py -u load_last_3_months.py
```

## Пример возможностей

Программа позволяет:
- загрузить курсы за сегодня;
- загрузить курсы на конкретную дату;
- посмотреть историю курсов в базе;
- посчитать средний курс за 3 полных месяца.
